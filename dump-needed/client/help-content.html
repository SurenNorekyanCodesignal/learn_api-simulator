Overview

API Simulator is a Postman-like client embedded in the task. You can send manual HTTP requests at any time, inspect responses, and optionally complete guided steps.

Send Requests

  1. Pick an HTTP method and enter a full URL.
  2. Use tabs to configure Params, Authorization, Headers, and Body.
  3. Press Send or use Ctrl/Cmd + Enter.
  4. Inspect status, timing, headers, and response body in the response pane.

For quick local testing, use the built-in demo API base URL: http://localhost:3001/demo-api.

Guided Steps

The left sidebar lists step goals. Selecting a step pre-fills the request editor. Some fields may be locked by the task. After you send, checks run automatically and completed steps are marked in the sidebar.

Troubleshooting

I get a network error

Confirm the API server is running on port 3001 and the URL is valid.

My JSON body fails

Use Body → JSON and make sure the payload is valid JSON syntax before sending.

Step did not complete

Check the step check results panel to see exactly which assertion failed.
